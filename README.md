# Retro2Cbot_web
web app for Retro2Cbot


## Backend
**how to build application**
1. cd backend
2. python3 -m venv venv 
3. 
- MacOs/Linux: source venv/bin/activate 
- Windows: venv\Scripts\activate
4. pip install -r requirement.txt
5. python app.py



## Frontend
**how to build app**
1. cd frontend
2. npm i 
3. npm run build
4. npm run start

**how to use app for dev purpose**
1. cd frontend
2. npm i 
3. npm run dev
4. npm run start:dev
